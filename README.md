# Saqartvelo-Mysteries
Short puzzle game created using Unreal Engine 5
# About Game
The game was intended as the final project for the course: Game Programming using UE5 and was created with the help of several game developers and designers.
Player navigates a compact world where puzzles are influenced by cultural elements. Each puzzle is designed to feel natural within the environment, rewarding observation and curiosity.
